# email-bomber
Floods an email address with hundreds/thousands of emails.

## Installation:

1. Make sure you have Node.js installed
2. Navigate to the root directory and install node-mailer
```
npm i node-mailer
```

3. Paste your outlook emails into emails.js
4. Run mail.js
```
node mail.js
```
